# Product Documentation: Premium Plan Features

## Overview
The Premium Plan is our most popular subscription tier, designed for power users who need advanced features and priority support.

## Key Features

### Storage and Capacity
- 1TB of cloud storage
- Unlimited file uploads
- Support for files up to 10GB each
- Automatic backup every 6 hours

### Collaboration
- Unlimited team members
- Real-time collaboration on documents
- Advanced permission controls
- Team activity dashboard
- Video conferencing integration

### Advanced Features
- AI-powered search across all documents
- Custom workflow automation
- API access with 10,000 requests per month
- Advanced analytics and reporting
- Custom branding options

### Security
- End-to-end encryption
- Two-factor authentication
- Single Sign-On (SSO) support
- Compliance certifications (SOC 2, ISO 27001)
- Advanced audit logs

## Pricing
$49 per user per month when billed annually
$59 per user per month when billed monthly

## Support
- 24/7 priority email support
- Live chat during business hours
- Dedicated account manager for teams of 10+
- Onboarding assistance
- Quarterly business reviews
